## Project Overview

### Description

Group Policy Preferences contained encrypted but recoverable credentials. The backup files contained sensitive information that should not have been accessible. Network traffic analysis revealed unencrypted communications containing user credentials. Lateral movement was facilitated by reusing discovered credentials across services.

## Features

### Key Features

- **hydra**: Privilege escalation was possible due to misconfigured sudo permissions.
- **feroxbuster**: The application stored sensitive credentials in an unencrypted configuration file.
- **impacket**: Privilege escalation was possible due to misconfigured sudo permissions.
- **nmap**: Network traffic analysis revealed unencrypted communications containing user credentials.
