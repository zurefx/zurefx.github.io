## Features

### Key Features

- **ffuf**: Initial enumeration revealed several interesting open ports on the target.
- **impacket**: The application stored sensitive credentials in an unencrypted configuration file.
- **nmap**: Token impersonation allowed elevation to SYSTEM privileges on the target.
- **feroxbuster**: The kernel version was outdated and vulnerable to a publicly known exploit.
