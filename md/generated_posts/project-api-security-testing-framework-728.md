## Project Overview

### Description

Lateral movement was facilitated by reusing discovered credentials across services. Network traffic analysis revealed unencrypted communications containing user credentials. The service account had excessive permissions assigned in Active Directory. Network traffic analysis revealed unencrypted communications containing user credentials.

## Features

### Key Features

- **hashcat**: Token impersonation allowed elevation to SYSTEM privileges on the target.
- **ffuf**: Unconstrained delegation was enabled on the compromised machine.
- **hydra**: Initial enumeration revealed several interesting open ports on the target.
- **gobuster**: The backup files contained sensitive information that should not have been accessible.
